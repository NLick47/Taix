# v1.2.0 更新日志

### 重构与优化
- 从单体桌面应用迁移为服务端、监听端并采用rust实现
- 客户端向.net10 + Aot 迁移
- Avalonia更新至 12.0.2
- 客户端首次采用Aot发布

### 客户端变化
- 去除开机自启设置
- 优化设置页操作逻辑
- 加入窗口渐变开关
- 应用启动默认不创建托盘



### 监控器
- 基于 Win32 API 实时监听前台窗口变化
- 新增**睡眠检测**：结合系统空闲时间与音频播放状态，精准识别休眠/锁屏
- 新增**音频状态检测**：专用 COM 线程轮询音频播放状态，防止误判睡眠
- 新增**崩溃恢复**：异常退出时从 `active_session.json` 恢复未上报的会话数据
- 支持 `config.json` 配置忽略进程列表.
- -

### 安装与部署
- 新增 `install.ps1`：创建开始菜单快捷方式（支持 Windows 搜索）并设置服务端与监控器开机自启
- 新增 `uninstall.ps1`：移除快捷方式、停止进程，仅删除可执行文件，保留用户数据
- 移除不再需要的 `organize-publish.ps1`
- CI 流程适配 AOT 发布、自动清理 PDB、修复手动触发时版本号问题
- 新增 `test-aot.yml` 用于独立的 AOT 构建验证

### 其他改进
- 清理 .NET 客户端与 Rust 服务的编译器警告
