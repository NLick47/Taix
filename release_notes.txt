## 下载与安装

本次发布提供以下产物，按需选择

| 产物 | 文件名 | 适用场景 |
| --- | --- | --- |
| Windows 安装器 | `Taix-{version}-Setup.exe` | 推荐普通用户使用，跟随cli指引安装 |
| Windows 绿色版 | `Taix-{version}-win-x64.zip` | 便携使用，需手动执行脚本注册开机自启（任务计划）服务 |
| macOS | `Taix-{version}-macos-arm64.dmg` | Apple Silicon 设备，macOS 12.0+ |
| 浏览器扩展 | `Taix-Web-Sentry-{ext_version}-chrome.zip` | Chrome / Edge 等 Chromium 浏览器，配合客户端使用以记录浏览数据 |

### Windows 安装器（推荐）

1. 下载 `Taix-{version}-Setup.exe`
2. 双击运行，按引导完成安装
3. 安装器会自动注册服务并启动，无需手动操作

### Windows 绿色版

1. 下载并解压 `Taix-{version}-win-x64.zip` 到任意目录（建议非系统盘）
2. 以**管理员身份**打开 PowerShell，进入解压目录
3. 运行 `install.ps1` 完成服务注册（脚本成功后会自动启动相关服务）
4. 卸载时同样以管理员身份运行 `uninstall.ps1`

> 绿色版升级：若从 1.2.0 及之后版本升级，可解压后直接替换 exe；或先在旧版本目录运行 `uninstall.ps1` 清理，再于新版本目录运行 `install.ps1`，请保留原图标目录与数据目录

### macOS

DMG 内含 `Taix.app`（客户端）和 `TaixTools/`（后台服务 + 安装脚本），两者都需要部署

1. 将 `Taix.app` 和 `TaixTools` 一起拖入 `Applications`
2. 终端运行 `/Applications/TaixTools/install-launchagent.sh` 注册后台服务
3. 启动 `Taix.app`

卸载运行 `/Applications/TaixTools/uninstall.sh`，Gatekeeper 拦截可在「系统设置 → 隐私与安全性」放行

### 浏览器扩展

- **Chrome / Edge**：解压 zip 后，在浏览器扩展管理页打开「开发者模式」，点击「加载已解压的扩展程序」选择解压目录
- **Firefox / Safari**：源码内提供对应 manifest，可自行打包加载

## 注意事项

**1. 时间格式变更说明**

自 v1.2.0 版本起，SQLite 本地存储已统一改为 UTC 时间，迁移后的数据将不再兼容旧版客户端，会存在时间偏移

**2. 从 1.2.0 之前的版本升级**

或其它迁移中遇到的问题，暂时移步到 #12

**3. 托盘说明**

应用托盘由 taix-shell 创建，修改托盘显示 / 隐藏需要重启该程序
