# v1.2.0 更新日志

### 重构与优化
- 从单体桌面应用迁移为服务端、监听端并采用rust进行重构，降低后台程序内存占用
- 客户端向.net10 + Aot 迁移，对客户端启动速度和内存占用做出优化
- Avalonia更新至 12.0.2
- 客户端首次采用Aot发布
- 优化休眠监控去除键盘、鼠标钩子采用更轻量级实现，性能损耗可以忽略不记
- sqllite前本地时间统一使用utc时间存储

### 客户端变化
- 去除开机自启设置
- 优化设置页操作逻辑
- 加入窗口渐变开关
- 应用启动默认不创建托盘
- 部分控件样式更改

### 注意事项
1. 本次更新涉及大规模重构，建议在升级前备份本地 SQLite 数据库文件，以防数据迁移异常
2. SQLite 本地存储已统一改为 UTC 时间，新版服务端会对旧数据进行迁移，迁移后数据不再兼容旧版客户端，会存在时间偏移
3. 已去除客户端开机自启设置，如需自启请手动添加至系统启动项
4. 需要使用powershell 运行 install.ps1 完成对taix-server、taix-monitor-windows 的服务安装，安装完成后默认会自动运行
